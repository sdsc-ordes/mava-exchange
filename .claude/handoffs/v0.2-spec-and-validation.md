## Gotchas / environment

- **Human-core files** (`spec/SPEC.md`, `spec/mava.ttl`, `spec/mava.shacl.ttl`,
  `src/mava_exchange/tracks.py`, `src/mava_exchange/__init__.py`) must not be
  edited without explicit user sign-off. Items 3‚Äì5 above all touch them ‚Äî flag
  and confirm before changing.
- **Pre-commit (`prek`)** runs `treefmt` + `ruff` over the **whole working
  tree** and stashes _unstaged_ changes; untracked files stay. Practical effects
  seen this session: a config ignore must be staged in the same commit as the
  file it covers; auto-fixers (trailing-whitespace, treefmt) modify files and
  fail the run ‚Äî just re-`git add` and recommit. Pre-format with
  `just format <paths>`.
- **Commit messages** end with the trailer:
  `Co-Authored-By: Claude Opus 4.8 (1M context) <noreply@anthropic.com>`.
- **Raw data** is gitignored under `data/{silent_child,tagesschau}/`. The
  extractor needs it; consumers of the example only need `examples/`.
- Dev commands:
  `just test | lint | typecheck | format | example | extract-examples | viewer`.
